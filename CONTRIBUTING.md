# Contributing to OpenFM

Thank you for your interest in contributing to OpenFM!

## Monorepo Structure

- `apps/web`: Next.js frontend application.
- `apps/api`: FastAPI backend service.
- `packages/engine`: Pure Python match simulator.
- `packages/db`: Alembic migrations and schemas.
- `packages/types`: Shared Typescript types.

## Development Setup

1. Install Node.js v20+ and Python 3.11.
2. Install pnpm: `npm i -g pnpm`.
3. Install dependencies: `pnpm install`.
4. Run dev: `pnpm run dev`.
